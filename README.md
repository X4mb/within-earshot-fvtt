# Within Earshot

Foundry **v14+** module: proximity voice over Simple Peer / WebRTC — remote volume follows token distance and walls, with optional GM global voice and a voice-token shortcut. Works with any game system.

**Install:** Setup → Add-on Modules → *Install Module* → paste:

`https://raw.githubusercontent.com/X4mb/within-earshot-fvtt/main/module.json`

For updates, use the same URL or install from a [release](https://github.com/X4mb/within-earshot-fvtt/releases) (`module.zip`).

**Develop:** `npm install` → `npm run build`. Release package: `npm run release-pack` (attach `module.zip` to a GitHub Release).
